# Basic HTML Website - Roadmap.sh Project

This is a multi-page personal portfolio website built as part of the [roadmap.sh](https://roadmap.sh/projects/basic-html-website) frontend beginner projects. The primary focus of this project is to establish a solid foundation in **Semantic HTML** and **SEO** without the use of CSS or styling.

## 🚀 Project Overview

The goal was to create a structured, accessible, and search-engine-friendly website that serves as a professional portfolio.

### Key Features
- **Multi-page Navigation:** Functional links between Home, Projects, About Me, and Contact pages.
- **Semantic Structure:** Utilization of HTML5 elements like `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` for better document hierarchy.
- **SEO Optimized:** Each page includes unique meta tags (title, description, keywords) and proper header nesting (`h1` through `h3`).
- **Clean Layout:** Used horizontal rules (`<hr>`) to simulate the logical separation of content as seen in professional wireframes.

## 🛠️ Technical Implementation

- **Language:** HTML5
- **SEO:** Metadata for indexing and mobile responsiveness.
- **Organization:** - `index.html`: Main landing page with experience and education summaries.
  - `projects.html`: Detailed list of major and minor projects.
  - `about.html`: Personal background and technical journey.
  - `contact.html`: Accessible form for user inquiries.

## 📂 How to Run
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Use the navigation links in the header to move between pages.
